Dockerfile

基本结构

Dockerfile是由一行行命令组成的。并且支持『#』开头的注释

  • 2355
  • 2355

一般而言，Docker分为四个部分：

  1. 基础镜像信息
    1. 222
    2. 333
  2. 维护者信息
  3. 镜像操作指令
  4. 容器启动时执行指令
# This dockerfile user the centos images
# VERSION 2 - EDITION 1
# Author: docker_user
# Command formart: Instruction [arguments/command]

# 第一行必须指定基础镜像
FROM centos

# 维护者信息
MAINTAINER docker_user docker_user@email.com

# 镜像的操作指令
